# Unreliable News Blocklist for Pi-hole

A Pi-hole compatible blocklist that blocks unreliable, biased, and conspiracy-driven news sources. The goal is to reduce exposure to low-quality news sources, across all devices on your network.

## Adding to Pi-hole

1. In the Pi-hole admin panel, go to **Lists**
2. Paste the URL ( https://raw.githubusercontent.com/OwlCaribou/FakeNewsBlocklist/refs/heads/main/fake_news.txt ) into the "Address" box.
3. Click "Add blocklist"
